JSON Web Tokens aka JWT.
JWT is an open industry standard (RFC 7519) for representing claims securely between two parties. 
@@NOTE:@@ Securely means that each party can guarantee that they are talking to each other and sent data is from an authentic source.
So the "sensitivity" of the data is NOT hidden.
JWT should be sent over HTTPS connections.
~JWTs are also good for ~Single-Sign-On applications, you can easily reuse the JWT.

~JWTs can be
*encrypted - to provide secrecy between parties, Not often used, a different RFC than ~JWTs.
*encoded - to verify the integrity of the claims contained as it is consumed by different apps. (ie special chars, binary, all properly represented)
When tokens are signed using public/private key pairs, the signature also certifies that only the party holding the private key is the one that signed it.
A session is when we "log in" (create a token) to when we "log out" (removal of a token).
JWT uses tokens to authenticate. It has these tokens stored in your web browser for the duration of the session.

The format of a JWT is: @@font-size:2em;color:MidnightBlue;header.payload.signature@@ 
*[[JWT Header]]
*[[JWT Payload]]
*[[JWT Signature]] - Signing data allows the data receiver to verify the authenticity of the source of the data.

		if (xhr.status == 404)

			alert(responseText);
		if (responseText.charAt(0) != '0')
			status = null;
		callback(status,params,responseText,url,xhr);
	};
	// do httpUpload
	var boundary = "---------------------------"+"AaB03x";	
	var uploadFormName = "UploadPlugin";
	// compose headers data
	var sheader = "";

bidix.UploadLog.prototype.addText = function(text) {
	if (!this.tiddler)
		return;
	// retrieve maxLine when we need it
	var maxLine = parseInt(config.options.txtUploadLogMaxLine,10);
	if (isNaN(maxLine))
		maxLine = -1;
	// add text
	if (maxLine != 0) 
		this.tiddler.text = this.tiddler.text + text;
	// Trunck to maxLine
	if (maxLine >= 0) {
		var textArray = this.tiddler.text.split('\n');
		if (textArray.length > maxLine + 1)
			textArray.splice(1,textArray.length-1-maxLine);
			this.tiddler.text = textArray.join('\n');		
	}
	// update tiddler fields
	this.tiddler.modifier = config.options.txtUserName;
	this.tiddler.modified = new Date();
	store.addTiddler(this.tiddler);
	// refresh and notifiy for immediate update
	story.refreshTiddler(this.tiddler.title);
	store.notify(this.tiddler.title, true);
};
